# Online Marketplace Management System (C++)

A **C++ console-based application** that simulates an online marketplace where sellers can list products and customers can browse, shop, and checkout.

This project demonstrates the **practical use of core Data Structures** in a real-world scenario.

## Features

- Seller & Customer Registration and Login
- Product Inventory Management
- Shopping Cart with **Undo** functionality
- Checkout Process with Receipt Generation
- Product Rating System
- Data Storage using Text Files
